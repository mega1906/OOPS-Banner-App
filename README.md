# OOPS-Banner-App

#### Clone the Repository
`git clone https://github.com/mega1906/OOPS-Banner-App.git`

#### Change the directory
`cd OOPS-Banner-App`

#### Create .gitignore file and edit it
`echo > .gitignore`   
`echo "*.class" >> .gitignore`

#### Commit
`git init`   
`git add .`   
`git commit -m "commit message"`   
`git push origin main`   

#### Compiler and Run java code
`javac OopsBannerApp.java`   
`java OopsBannerApp`

#### Checkout and Merge
`git checkout -b <branch-name> `   
`git add . or <file-name>`     
`git commit -m "commit-message"`    
`git push origin <branch-name>`      

`git checkout <branch-name>`    
`git pull origin <branch-name>`    
`git merge <merge-branch-name>` or `git merge -X theirs <merge-branch-name>`          
`git push origin <branch-name>`   

#### Switch branches
`git switch <branch-name>`  

#### Delete a branch
`git branch -d <branch-name>` or `git branch -D <branch-name>`
